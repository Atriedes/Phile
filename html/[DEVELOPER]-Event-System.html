One of the main feature of Phile is new event system, which replace the hook system of pico.

In the core we trigger a lot of events, and for every event you can register your plugin to get informed.

To register your plugin for an event, you only need to call a static method in your plugin constructor, like this:

Phile\Event::registerEvent('before_render', $this);

With this line of code, you register your plugin for the event with the name "before_render".

The second important step is to implement the interface \Phile\EventObserverInterface. This interface define only one method: public function on($eventKey, $data = null). These method will be called from the event system to notify your plugin about an event. We have added an example plugin in the core, please look at the PhileDemoPlugin in the folder plugins/phileDemoPlugin.

You can find a list of all triggered events on the startpage of the demo pages after installing Phile.

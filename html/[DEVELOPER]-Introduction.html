In this wiki we have collect some informations for developers. please refer to the pages prefixed with [DEVELOPER]

If you plan a change at the core, please create first an issue an let us discuss this change, before you do any changes. Not all changes will be accepted by the maintainer of the project, you can save a lot of time if you talk to us, before start coding.

This project is based on Pico CMS v0.8 by Gilbert Pellegrom.

The desire to fork the project from Pico, came when a few community members wanted to contribute more and increase the rate of development progress.

The first step was a complete refactoring to use the full power of object oriented programming (OOP). The Core and all components are now object oriented and use modern concept of programming like some design patterns.

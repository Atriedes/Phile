The new plugin concept of Phile is a little bit different to pico, but also easy to understand. A lot of logic follows the concept convention over configuration, so please read the following list of conventions.

  • Your Plugin need an own folder under the plugins/ folder
  • The name of the folder is the name of your class, but first character has to be lowercase!
  • The class name of your plugin is the name your plugin folder, but first character has to be uppercase!
  • Within your plugin folder you need at least one file with the name plugin.php
  • Within your plugin folder you can add a file with the name config.php, this file must return an array.
  • Your plugin class must extend the \Phile\Plugin\AbstractPlugin class.
